# การผลิตและการใช้พลังงานไฟฟ้าส่งผลต่อปริมาณการปล่อย CO₂ ของประเทศไทยมากขึ้นหรือไม่

การปล่อยก๊าซคาร์บอนไดออกไซต์ (CO2) เป็นปัญหาสำคัญที่ส่งผลกระทบต่อสิ่งแวดล้อมและภูมิอากาศของทั่วโลก เนื่องจาก CO2 เป็นก๊าซเรือนกระจกที่สะสมอยู่ในชั้นบรรยากาศทำให้เกิดภาวะโลกร้อน (Global Warming) ซึ่ง สาเหตุหลักของการเกิดก๊าซดังกล่าวมาจากพฤติกรรมของมนุษย์ 
จากข้อมูลการปล่อยก๊าซเรือนกระจกของ Climate Watch Platform พบว่า ประเทศไทยมีปริมาณการปล่อยก๊าซเรือนกระจกเพิ่มขึ้นในทุกๆ ปี (Figure 1) ซึ่ง ส่วนใหญ่มาจากภาคพลังงาน (Figure 2)
